# spooky_connect4

Rust and Python library for the game of Connect4.

# Rust

```fish
cargo add spooky_connect4
```

# Python

```fish
uv add spooky-connect4
```

## Develop

### Tests

- `fish run_tests.fish`
    - `fish run_python_tests.fish`
    - `fish run_rust_tests.fish`

### Lints

- `fish run_lints.fish`
